## 🎯 使用方法

### 启动应用
```bash
python comic_translator_app.py
```

### 配置API（首次使用）
1. 应用启动时会检查API配置
2. 如未配置，会提示打开设置
3. 选择API服务商并输入密钥
4. 选择要使用的模型
5. 保存配置即可使用

### 切换配置（日常使用）
1. 点击主界面的 **"设置"** 按钮
2. 在设置窗口中修改配置
3. 点击 **"保存"** 应用更改
4. 配置立即生效，无需重启

### 重置配置（出现问题时）
1. 在设置窗口点击 **"重置默认"**
2. 确认重置操作
3. 所有配置恢复到初始状态
